# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues privately via [GitHub Security Advisories](https://github.com/zerckzzyHD/Robco-UOS/security/advisories/new).
Include a description of the issue, steps to reproduce, and the potential impact.
The maintainer will acknowledge the report and respond as quickly as possible.

Public disclosure should be coordinated after a fix is available.
